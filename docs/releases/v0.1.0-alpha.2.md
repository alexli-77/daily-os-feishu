# v0.1.0-alpha.2

Daily OS Feishu alpha.2 focuses on making the local alpha easier to run, inspect, and tune before using it on another Mac.

## Highlights

- Added the macOS desktop companion prototype with the penguin floating entry point.
- Added hover-based daily todo reminder support for the companion.
- Added Daily Todo Inbox capture so casual user reminders can feed daily plan and review context.
- Added the dashboard Decision Policy page for editing `decision-policy.md`.
- Added a plain Guide page in the dashboard with the Feishu commands and setup boundaries.
- Added the weekly-review skill bridge to call `life-review-os` for deeper weekly planning and Feishu Weekly write-back.
- Improved Feishu message card formatting for daily plan, weekly review, skill output, and write-back confirmation.
- Added support for Claude Code CLI as an additional LLM provider while keeping workflow output formatting unified.

## Validation

Run these before publishing the GitHub prerelease:

```bash
npm run typecheck
npm run build
npm run regression:test
npm run privacy:scan
npm run alpha:smoke:ci
```

## Install Boundary

This is still a local-first alpha. It supports one-command service installation with `npm run service:install`, but it is not yet a signed or notarized macOS app bundle.

For another Mac, use the dashboard Guide page and `docs/first-install-checklist.zh-CN.md` during setup.
